- 내가 푼 문제를 기반으로 내가 자주 틀리는 유형, 많이 풀었지만 보충할 유형 등 내가 풀면 좋은 문제들을 추천해줍니다.
- 문제를 풀고 내가 푼 문제를 정리하고 다른 사람들과 문제 풀이 방법을 공유하며 실력을 향상시킬 수 있습니다.
